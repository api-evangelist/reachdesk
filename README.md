# Reachdesk

Reachdesk is a global B2B corporate gifting and direct mail platform that enables sales, marketing, and customer success teams to send physical gifts, branded merchandise, and digital rewards at scale. The platform integrates with 20+ CRM, marketing automation, and sales engagement tools and reaches recipients across 180+ countries.

## API

The Reachdesk Trigger Campaign API allows programmatic sending of gift campaigns. API tokens are generated in the Reachdesk platform under Organization > Settings > API Tokens. The API is documented in the Reachdesk Knowledge Base and supports integration with tools like Salesforce, HubSpot, Marketo, Outreach, and Zapier.

## Plans

Plans start at $20,000/year with three tiers (Core, Plus, Premium) differing in user count, team count, warehouse locations, and geographic reach. All pricing is custom-quoted via direct sales.

See [plans/reachdesk-plans-pricing.yml](plans/reachdesk-plans-pricing.yml) for full details.

## Rate Limits

Rate limits for the Reachdesk API are not publicly documented. Monthly gift volume allocations per plan tier serve as a practical throughput limit. See [rate-limits/reachdesk-rate-limits.yml](rate-limits/reachdesk-rate-limits.yml).

## FinOps

Cost categories include platform license, gift/merchandise costs, swag sourcing, shipping/fulfillment, and API usage (included in subscription). See [finops/reachdesk-finops.yml](finops/reachdesk-finops.yml).
